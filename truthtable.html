Truth Table Generator

This page contains a JavaScript program which will generate a truth table given a well-formed formula of sentential logic. You can enter multiple formulas separated by commas to include more than one formula in a single table (e.g. to test for entailment). Tables can be displayed in html (either the full table or the column under the main connective only), plain text, or LaTex. You can also select which pair of symbols to use for the two truth values.

Select Type of Table:

Symbols: use the following symbols for the various logical connectives:

~for negation
&for conjunction
vfor disjunction
>for the conditional
< >for the biconditional
#for absurdity/falsum
|for the Sheffer Stroke, aka NAND

Here are some examples of well-formed inputs the program will accept:

  • ~A
  • (A & B)
  • (# > (B v ~A))
  • (A & (~B > C))
  • (A<>(BvC)), A, (~B>C)
